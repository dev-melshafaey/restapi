# restapi
RESTful api using slim framework 

### Usage

### Installation

import database from slimapp.sql

Edit db/config params

Install SlimPHP and dependencies

$ composer
API Endpints

$ GET /api/customers
$ GET /api/customer/{id}
$ POST /api/customer/add
$ PUT /api/customer/update/{id}
$ DELETE /api/customer/delete/{id}
